# 🧠 Annotation Bench: AI Trainer Practice Tool

A self-directed, hands-on practice tool built to simulate real-world AI annotation and AI trainer tasks. Created as part of a personal learning journey to transition into the AI data annotation field.
https://freyton7.github.io/AI-Annotation-Practice-Task/
---

## 📌 About This Project

This project was built to practice and demonstrate core AI annotation skills without formal employment experience. It covers the types of tasks commonly required in AI data labeling and RLHF (Reinforcement Learning from Human Feedback) workflows.

---

## ✅ What It Covers

- Text classification and sentiment labeling
- Intent tagging and response categorization
- Model output ranking and evaluation
- Prompt quality assessment
- Annotation accuracy exercises with guidelines

---

## 🎯 Purpose

This tool was created to:
- Build hands-on annotation skills independently
- Simulate real AI trainer workflows
- Demonstrate initiative and readiness for AI annotation roles

---

## 🛠️ Built With

- HTML / CSS / JavaScript
- Claude AI (as learning and development platform)
